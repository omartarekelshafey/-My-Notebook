# CH 3 Introduction to Windows   Event Logs

## Windows Event Logs Overview

تعتبر EVENT LOGS  في نظام ويندوز هي "الصندوق الأسود" لأي محقق جنائي رقمي أو محلل أمني. من غيرها، إحنا حرفياً ماشيين في الضلمة. logs دي بتسجل كل كبيرة وصغيرة بتحصل في النظام، ومكانها الافتراضي بيبقى في المسار التالي:

`C:\Windows\System32\winevt\Logs`

***

## Windows Event Types

#### Security Event Log

ده أهم نوع (The Holy Grail) لمحلل الـ SOC، ومنه بنقدر نتابع تحركات المستخدمين:

* ال Logon events: بتسجل عمليات الدخول والخروج، سواء نجحت أو فشلت.
* ال Account management: بتراقب أي تغيير في الحسابات زي إنشاء مستخدم جديد أو تغيير باسورد.
* ال Object access: بتعرفنا مين فتح ملفات معينة أو دخل على فولدرات مشتركة.
* ال Process tracking: بتسجل كل "بروسيس" بتبدأ وتخلص، وده أساسي لاكتشاف البرامج الخبيثة.

#### System Event Log

بيهتم أكتر باستقرار النظام، بس فيه معلومات أمنية ذهبية:

* حالة الـ Services: لو فيه خدمة فجأة وقفت أو خدمة غريبة اشتغلت.
* حالة الـ Firewall: بيسجل لو جدار الحماية اتقفل أو حصل فيه تغيير.
* عمليات Startup/Shutdown: عشان نعرف الجهاز رستر إمتى وبسبب إيه.

#### Application Event Log

بيحتوي على logs الخاصة بالبرامج المتسطبة، زي الأنتي فيروس (Antivirus) أو قواعد البيانات، وده بيساعدنا نعرف لو التطبيقات حصل فيها تلاعب.

#### Other Event Log Types

ويندوز فيه أكتر من 300 نوع logs تانيين، بس أهمهم لينا كأمنيين:

1. ال PowerShell logs: بتكشف الأوامر اللي الهاكرز بيكتبوها (Scripting attacks).
2. ال Scheduled tasks logs: عشان نكتشف لو فيه "تاسك" خبيثة محطوطة عشان تشتغل لوحدها.
3. سجلات RDP: لمراقبة الاتصالات عن بُعد.

## Windows Event Log Analysis Tools

مش دايما بنعتمد على الـ SIEM، ساعات بنحتاج أدوات تانية

| **TOOL**            | **TYPE TOOL**   | **USAGE**                                              |
| ------------------- | --------------- | ------------------------------------------------------ |
|  Event Viewer       |  (Built-in)     | استعراض السجلات بسرعة على الجهاز الحي.                 |
|  PsLogList          | Sysinternals    | سحب السجلات وتحويلها لصيغ زي CSV أو XML.               |
|  Event Log Explorer |  (GUI)          | تحليل ملفات `.evtx` أوفلاين بفلترة قوية جداً.          |
|  EvtxECmd           | CLI (Zimmerman) | أداة سريعة جداً للمحققين لتحويل السجلات لتقارير منظمة. |
