# Tools and Services

## The Harvester

&#x20; هي عباره عن تول  بنسخدمها علشان نجمع  ip او ايميلات عن دومين معين&#x20;

لو عايز تجمع بيانات عن **Google**، هتستخدم الأمر ده:

```bash
theharvester -d google.com -l 100 -b google
(tool) (target domain = google.com) (list 100 results max) (source = google)
```

هنا مثلا انا عايز اجيب معلومات عن الدومين جوجل  والمصدر اللي ادور فيه جوجل&#x20;

```bash
theharvester -d google.com -l 100 -b linkedin
```

هنا مثال تاني عايز اجيب معلومات عن دومين جوجل بس من خلال لينكدان مش جوجل&#x20;

## Maltego

Following the line of TheHarvester, we will now have a very powerful OSINT tool called Maltego.

Maltego is a high-level data mining and information gathering tool, capable of obtaining real-time data on different types of entities (companies, people, websites, etc.), and representing them graphically through nodes, showing all the connections that the program was able to obtain over the Internet, about the subject under investigation.

دي مقارنة بين **Maltego** و **Harvester** في جدول:

| **الخاصية**               | **Maltego**                                                    | **Harvester**                                                      |
| ------------------------- | -------------------------------------------------------------- | ------------------------------------------------------------------ |
| **نوع الأداة**            | أداة تحليل جنائي وتحقيقات سيبرانية                             | أداة جمع بيانات مفتوحة المصدر                                      |
| **الغرض الرئيسي**         | تحليل العلاقات بين الكيانات وجمع البيانات المتقدمة             | جمع بيانات أولية مثل البريد الإلكتروني                             |
| **البيانات المدعومة**     | الأشخاص، الشركات، المواقع الإلكترونية، العناوين IP، وغيرها     | عناوين البريد الإلكتروني، DNS، من محركات البحث والشبكات الاجتماعية |
| **إمكانية التحليل**       | تحليل شامل وعلاقات بين الكيانات بشكل رسومي ومرئي               | جمع البيانات دون تحليل العلاقات                                    |
| **دعم مصادر البيانات**    | يدعم العديد من المصادر المتنوعة مثل DNS، WHOIS، Shodan، وغيرها | محركات البحث، الشبكات الاجتماعية، DNS                              |
| **التحليل المرئي**        | نعم، يعرض العلاقات بشكل رسومي وواضح                            | لا، لا يوجد تحليل مرئي أو رسومي                                    |
| **التكامل مع أدوات أخرى** | يدعم التكامل مع أدوات أخرى مثل VirusTotal، Shodan، وغيرها      | لا يدعم التكامل مع أدوات أخرى                                      |
| **التخصيص**               | عالي، يسمح بتخصيص العمليات وتحليل العلاقات بشكل عميق           | محدود، يقتصر على جمع البيانات فقط                                  |
| **التطبيقات العملية**     | تحليل الشبكات، التحقيقات السيبرانية، البحث الجنائي             | جمع معلومات أولية، تقييم الأهداف المحتملة                          |
| **المجانية**              | النسخة المجانية محدودة، يوجد إصدار مدفوع متقدم                 | مجانية ومفتوحة المصدر                                              |



## Defending Against Google Dorks

علشان احمي نفسي من dorking بعمل كذا حاجه منهم geofencing, IP Whitelisting

ايه ال Geofencing&#x20;

الـ **Geofencing** هو وسيلة بتستخدمها بعض المواقع أو السيرفرات علشان **تمنع الوصول من دول معينة**. يعني مثلًا، لو شركة بريطانية عايزة موقعها يكون بس للناس اللي موجودين في بريطانيا، ممكن تستخدم Geofencing علشان **تحظر كل الـ IPs اللي جاية من برّه بريطانيا**.

بس في مشكله هنا \
انه أي حد في أمريكا أو أي دولة تانية يقدر يستخدم **VPN**، ويخلي اتصال الإنترنت بتاعه يخرج من سيرفر موجود في بريطانيا.\
فالموقع هيفتكر إن الشخص ده فعلاً في بريطانيا، ويفتح له المحتوى!&#x20;

علشان احل مشكله ديه بعمل كده

| الوسيلة             | بتحمي من إيه؟                                      |
| ------------------- | -------------------------------------------------- |
| VPN Detection       | المستخدم اللي بيستعمل VPN علشان يزوّر موقعه        |
| Behavioral Analysis | أي سلوك مش طبيعي حتى لو الـ IP شكله تمام           |
| 2FA                 | بيثبت الهوية حتى لو الشخص وصل للموقع بطريقة ملتوية |

&#x20;**IP Whitelisting**

"أنا هاسمح بس للـ IPs دي تدخل، والباقي كله بلوك، حتى لو عارف الباسورد."
